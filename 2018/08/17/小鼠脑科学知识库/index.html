定义

结构化的语义知识库，描述实体间的相互关系，基本单位为 SPO 三元组。

构建流程

  1. 知识抽取
    • 实体抽取：即命名实体识别（NER）
      • 基于词典和规则
      • 基于统计机器学习
        • 有监督学习方法：SVM, HMM, ME, CRF
        • 半/弱监督学习方法：预训练+特征融合
      • 基于深度学习的方法：Bi-LSTM+CRF
    • 关系抽取
      • 基于特征向量或核函数的有监督关系分类
      • 无监督聚类
    • 属性抽取/事件抽取
      一般可转为关系抽取

  2. 知识融合

    • 实体链接：从文本中抽取得到的实体链接到知识库中对应的正确实体对象
      • 实体消歧
        基于包含语义信息的特征向量模型（词袋模型）的聚类
      • 共指消解
        计算实体相似度然后进行聚类
    • 知识合并
      • 合并关系型数据库
        RDB2RDF
      • 合并外部数据库
        数据层的融合和模式层的融合
  3. 知识加工
    • 本体构建
      人工编辑手动构建，辅以统计机器学习算法自动抽取上下位关系（IsA 实体对）
    • 知识推理
      • 基于逻辑的推理
        基于本体概念层（OWL）的推理如 SWRL
      • 基于图的推理
        神经网络模型或 Path Ranking 算法

一个小鼠脑科学知识库的初步搭建流程和技术路线

知识抽取

  • 半结构化数据
    Wikipedia 中的脑区核团信息框：利用 Wikidata api 提取json键值对存入数据库

  • 非结构化数据

    • 参考 Allen 脑解剖学命名树和 Wikipedia 上的核团介绍构建实体关系表、同义词表，构建基本的 SPO 三元组形成关系网，用图数据库neo4j存储并展示

    • 对 pubmed 摘要和全文作自然语言处理补充关系网
      工具：spacy, Stanford CoreNLP

      • 分句，分词，词形还原（lemmatization），词性标注
      • 关键词提取，N-grams，tf-idf
      • 句法分析（parsing）和依存路径解析
      • NER
        1. NeuroNER
        2. Bi-LSTM+CRF
          参考论文
          Bidirectional LSTM-CRF Models for Sequence Tagging

      • NRE

        1. 共现关系抽取
        2. 基于深度学习BiLSTM的关系分类
        3. snorkel弱监督方法，辅以生物医学本体库的远程监督

知识融合

  • 构建领域知识本体
    • 构建工具：protégé
    • 参考本体资源
      • 鼠脑结构：Allen鼠脑图谱本体
      • 鼠脑模拟信息：成年小鼠解剖学本体
      • Biolink
    • 手工构建流程
      将核团实体和之间的关系抽象得到本体类，定义分类概念和层级，用 protégé 填充相关联的类别、关系、属性等，确定数据属性和对象属性的取值。然后把本体交由具有神经科学相关知识的研究人员审核。
  • 本体匹配
    • 模式匹配
    • 实例匹配

知识存储

  • 存储方式
    关系型数据库 MySQL或文档型数据库MongoDB(用 OWL 形式化表达 ER 图(D2RQ))
    图数据库 Neo4j
  • 分布式计算框架(可选)
    Hadoop

形式化展示

  • web 服务框架
    使用 ElasticSearch（动态） 或 Apache Solr（静态）构建
  • 信息展示
    1. 按照 anatomy(structure, neuron types, input and output, neurochemistry), function, clinical significance三部分，建立类似于LBD的直观连接图，以链接数据的形式展示脑区结构、功能和关联疾病间的关系。
    2. PubMed文献实体和关系标注
  • 对外服务
    • 建立查找索引，直接搜索关键词
    • 使用 SPARQL 查询 RDF
