林轩田机器学习基石笔记（二）

发表于 2018-08-23
字数统计 913 | 阅读时长 4

这篇笔记是林老师机器学习课程第三部分的内容：How Can Machine Learn?
主要涵盖三个线性分类模型：linear classification、linear regression、logistic regression，和一个非线性变换 nonlinear transformation。详细内容如下：

Linear Regression

解决曲线拟合问题。设输入 X 为 d 维，加上常数项维度为 d+1，与权重w的线性组合即为 Hypothesis，记为h(x)，即 $h(x) = w^T * X$，误差为均方误差。

最小化$ E_{in}(w) = \frac{1}{N} * \sum_{n=1}^N (w^Tx_n-y_n)^2$，利用线性最小二乘法的求解方法（矩阵求解或者迭代法随机梯度下降）求解得到权重向量 $ w = (X^T X)^{-1} X^TY $，一般情况下，只要满足样本数量N远大于样本特征维度d+1，矩阵就是非奇异的（可逆的）。然后证明了同 linear classification 类似，当N足够大的时候，抽样样本的误差值 $E_{in}$和总体样本的误差值$E_{out}$都收敛于平均噪声，满足VC Dimension，可以机器学习。

Logistic Regression

二分类问题。使用将特征加权和 $ s = \sum_{i=0}^dw_ix_i $限制在 [0,1] 间的 Sigmoid Function $\theta(s) = \frac{1}{1+e^{-s}}$，带入得到 hypothesis 为 $h(x) = \theta(w^Tx) = \frac{1}{1+e^{-w^Tx}}$。
其误差函数用cross­-entropy error表示，最小化$ E_{in}(w) = \frac{1}{N} * \sum_{n=1}^N ln(1+exp(-y_nw^Tx_n)) $，最后还是通过梯度下降法计算$ \nabla E_{in}(w) \approx 0$时的对应的 w 值。
交叉熵的介绍见这篇文章：关于交叉熵在loss函数中使用的理解

最后本节介绍了用线性分类的方法求解多分类问题的两种方法：One­Versus­All(OVA)和One­Versus­One(OVO)。对 OVA，分成多个计算二分类问题，用逻辑回归计算每个点属于各类的概率，取最大概率的类别即可。OVO 主要为了解决类别很多时，正负数据不平衡的问题。它每次只取两类进行binary classification，按照单个点被分成某类的次数排序，取次数最多的即可。

Nonlinear Transformation

本节介绍用非线性的模型来解决分类问题。思路是用特征转换（Feature Transform）把线性不可分的x空间的点特征映射到z空间中去，在 z 空间完成线性分类后再用 x 的多项式替换。例如对于二次的 hypothesis，它包含二次项、一次项和常数项1，即$\phi_2$(x) = (1,$x_1$,$x_2$,$x_1x_2$,$x_1^2$,$x_2^2$)，应使z域的维度与x域中多项式的个数一致，即样本特征维度 d+1 = 6维。
非线性变换带来的问题是时间复杂度和空间复杂度的增加，所以应在 $E_{in}$满足条件的情况下尽量选用最简单的模型。
